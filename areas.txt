REPOSITORY: (committed)
(directorio git)




Modified:
    Si la versión del archivo contiene cambios que no son parte del repositorio
    Y NO se ha añadido al staging AREA

Staged:
    Si la versión del archivo contiene cambios que no son parte del repositorio
    Y SI se ha añadido al staging AREA

Committed:
    YA se añadió al directorio git




Commit: 
    Es un registro o foto del estado de un proyecto en un momento especifico
    Registra los cambios que se realizaron en un archivo
    Cada commit añada un SHA (identificador único)



Branch:
    Linea independiente de desarrollo en un repositorio, sin afectar la que está en vivo








Comandos:

git add <file>
git rm --cached <file>..." to unstage
git commit -m "texto"

Para cambiar el editor por defecto
git config --global core.editor "code --wait"
